## Installation

### Cloner le projet

```shell
git clone https://github.com/Bts-Sio-CCI/FlowTechAP2.git
```

### Installer le compilateur SASS

```shell
npm install -g sass
```

## Utilisation

### Compiler le SASS

```shell
sass --watch ./scss/custom.scss ./css/custom.css
```

ou utilisez la commande raccourci:

```shell
npm run csscompile
```
